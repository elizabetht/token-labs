Methodology

Hardware

  • Platform: NVIDIA DGX Spark
  • GPU: Grace Hopper (~120GB GPU memory)
  • Architecture: ARM64
  • TDP: ~150W per node

Benchmark Configuration

  • Prefill Test: 100 prompts, 3072 input tokens, 1024 output tokens (3:1 ratio)
  • Cache Test: Enabled (LMCache with CPU offload) - 100 prompts with prefix repetition
  • Decode Test: 100 prompts, 1024 input tokens, 3072 output tokens (1:3 ratio)
  • Request Rate: 10 requests/second
  • Tool: vllm bench serve

Metrics

  • Interactivity (TPS/User): Output tokens per second per user — derived from decode throughput / concurrent request count. Higher = more responsive user experience.
  • Throughput (TPS/kW): Total output tokens per second / node power draw (0.15 kW). Higher = more revenue per unit of power.
  • Cost: (DGX hourly cost × 1,000,000) ÷ (tokens/sec × 3600). DGX Spark: $4000 ÷ 26,280 hours (3yr @ 30% utilization) + electricity ≈ $0.07/hour
